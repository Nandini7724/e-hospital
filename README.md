# e-Hospital Management System

Java, MySQL, Swing

A desktop-based Hospital Management System developed using Java Swing and MySQL to manage hospital operations efficiently.

## Features

- User Login Authentication
- Patient Registration
- Search and Update Patient Records
- Doctor Management
- Appointment Booking
- Billing System
- Medicine Inventory Management
- Dynamic Dropdowns for Better Usability
- MySQL Database Integration

## Tech Stack

- Java
- Java Swing
- MySQL
- JDBC
- IntelliJ IDEA

## How to Run

1. Open project in IntelliJ IDEA
2. Start MySQL server
3. Run `LoginPage.java`
4. Login using admin credentials
